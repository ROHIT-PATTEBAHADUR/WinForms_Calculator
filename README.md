🧮 Windows Forms Calculator

A simple and user-friendly calculator application built using C# and Windows Forms in Visual Studio. This project performs basic arithmetic operations with a clean graphical interface.  

🚀 Features  
➕ Addition  
➖ Subtraction  
✖️ Multiplication  
➗ Division  
🧹 Clear (Reset) functionality  
🖥️ Simple and intuitive UI  
🛠️ Technologies Used  
C#  
Windows Forms (.NET)  
Visual Studio

⚙️ How to Run the Project  

Clone the repository:  

git clone https://github.com/your-username/WinForms_Calculator.git  
Open the project in Visual Studio  

Build the solution:  

Build → Build Solution  

Run the application:  

Press F5  
📂 Project Structure  
WinForms_Calculator/  
│
├── Form1.cs          # Main logic of calculator
├── Form1.Designer.cs # UI design code
├── Program.cs        # Entry point
├── bin/              # Build output (ignored)
├── obj/              # Temporary files (ignored)
